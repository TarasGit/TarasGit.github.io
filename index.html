Hier befinden sich kurze Beschreibungen und Links zu den einzelnen Projekten

Die meisten Projekte sind im Zusammenhang mit dem Informatikstudium entstanden.

  • Diplomarbeit an der Humboldt-Universität Berlin

    Empirische Studie verschiedener Algorithmen für das Multi-Objective Release Planning

  • React/Flux - Todo-List

    Diese Webapp implementiert eine Todo-List, sodass die CRUD-Operationen ereignisbasiert mittels Flux-Design-Pattern durchgeführt werden.

  • Nodejs/React-ImageGallery

    Hier wurde eine Webapp erstellt, welche die Möglichkeiten mitbringt die Bilder auf den Server von einem Client hochzuladen, sodass diese in Kachelform angezeigt werden.

  • Tanks90x - Java

    Diese App ist als Spaß entstanden und implementiert ein grafisches Spiel aus den 90er Jahren.

  • Asynchrone Suche - OpenLib

    Diese Webapp such mittels einer REST-API auf dem Server nach bestimmten Daten.

  • Buddy-List - Android

    Diese App implementiert eine Bluetooth-Suchfunktion auf einem Android-Smartphone, welche die gepaarten Buddies (Freunde) in der Umgebung scannt und grafisch auflistet.

  • PostgreSQL Buffer - C

    Das ist ein Projekt, welches die Buffer-Funktionalität im PostgreSQL umsetzt damit die Tuples nicht einzeln sondern gepuffert geladen werden, sodass die Produktivität der Anfrageverarbeitung erhöhrt wird.

Studienarbeit - USB GadgetFS on Android

Die Studienarbeit beschäftigt sich damit einen GadgetFS-Treiber im Kernel auf einem Android-Galaxy-Smartphone einzurichten, sodass eine Möglichkeit geschaffen wird die USB-Verbindung im Gadget- oder im Host-Modus zur Laufzeit zu konfigurieren.[Präsentation zu der Studienarbeit]
